# Cache Controller

## Project Overview
This project implements a Cache Controller using Verilog HDL. The controller manages cache read and write operations, detects cache hits and misses, and communicates with the main memory to improve data access efficiency.

## Features
- Cache hit and miss detection
- Read and write operations
- Memory interface control
- RTL simulation and functional verification
- Synthesizable Verilog design

## Tools Used
- Verilog HDL
- Synopsys VCS
- Synopsys Design Compiler
- IC Compiler II
  

## Project Files
- cache_controller.v
- cache_controller_tb.v
- Supporting Verilog modules (if applicable)

## How to Run
1. Compile the Verilog source files.
2. Run the testbench using Synopsys VCS.
3. Verify the simulation results using the generated waveforms.

## Applications
- Processor cache systems
- Memory hierarchy design
- Digital ASIC and FPGA design
